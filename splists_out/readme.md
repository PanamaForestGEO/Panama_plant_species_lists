#Species List Processing Outputs

This subdirectory contains processed species lists, including merged datasets and outputs from taxonomic resolution (e.g., WCVP).

- **alternates_and_synonyms/**: Contains lists of "alternate names" (previous morphospecies labels) used before newer identification. Includes source-specific files and a combined AltNames file.

- **needs_check/**: Species lists requiring botanical revision due to missing records or naming conflicts between datasets.

- **Plantnet/**: A consolidated list of vascular plants in BCI/BCNM, created by merging multiple botanist-reviewed lists.

- **species_list_WCVP/**: Species lists updated with standardized WCVP nomenclature following taxonomic resolution and manual quality control. These changes have not been reviewed by a botanist, should be used carefully. 

- **zarchive/**: Deprecated or legacy versions of species lists.
