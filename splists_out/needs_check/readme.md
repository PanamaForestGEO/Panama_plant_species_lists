# Species Lists for Botanical Review

These lists were generated by the `plantnet` script while merging various sources to create a consolidated BCNM list. These entries require expert verification.

### Woody Status Check
* **`BCI_SPECIES_LIST_POSSIBLY_WOODY`**: Species in the BCNM botanist list but absent from the current Panama woody list; flagged by lifeform as "might_be_woody"

### Discrepancy Lists (Unmatched Species)
* **`lianasbci_not_garwood`**: Lianas present in BCI records but missing from the Garwood reference.
* **`plotsbci_not_garwood`**: Species identified in BCI/BCNM plots missing from the Garwood list.
* **`zotz_garwood_check`**: Species from the Zotz list that lack a definitive match in Garwood or have questionable match quality.
* **`Panamabiota_species_no_garwood`**: Species in the PanamaBiota list missing from the BCNM botanist list.
* **`Wright_not_CurrentPanamaWoody`**: Species in the Wright dataset missing from the current Panama Woody list.
