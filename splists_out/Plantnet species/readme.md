# PlantNet Species List for Plant Net

## File Registry
- **BCI_PLANTS_panamabiota_WCVP**: Panama biota BCI list (Angios/Monocots/Ferns) matched to WCVP.
- **BCI_PLANTS_PANAMABIOTA_WCVP_changes**: Panama biota BCI list with taxonomic updates with WCVP.
- **PANAMA_BIOTA_BCI**: Panama biota BCI list with woody species indicators and name mapping.
- **WOODY_SPECIES**: Forestgeo Woody list with BCI plot presence indicators and name mapping.
- **GARWOOD**: Garwood list with wcvp names and original columns. And columns indicating presence in panama biota and woody list. ( the lists above also have a column indicating Garwood presence).
- **GARWOOD_PANAMA_BIOTA_CLOSE_MISMATCH**: list of 40 species in Garwood that match at the species level to Panama biota but not the subspecies or variety.
- 
*Note: Excel files contain a second sheet with a full Data Dictionary.*
