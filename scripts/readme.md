# Scripts

- **checkandmergesplists.r**  
  Older version of how to check consistency of woody species lists.

- **checkandmergesplot.Rmd**  
  Newer version of how to check consistency of woody species lists.  
  Creates a report that needs to be saved in the reports folder.

- **compare_tnr_wcvp.html**  
  Work in progress by Paula U; the idea is to see the mismatches produced by TNRS and WCVP.  
  (I suspect TNRS can make more mistakes.)

- **species_plantnet.Rmd**  
  Creates a combined list of species in BCI by joining Garwood, Zotz, and Panamabiota species.

- **wcvp_matching_garwood**  
  Script to add WCVP names to the Garwood list.

- **wcvp_matching_lianas**  
  Script to add WCVP names to the lianas list.

- **wcvp_matching_lianasHM**  
  Script to add WCVP names to the lianas list. Modifications by HM.

- **wcvp_matching_panamabiota**  
  Script to add WCVP names to the Panamabiota list.  
  Joins different Panamabiota sources.

- **wcvp_matching_wrights**  
  Script to add WCVP names to the Wright list.  
  This list is useful for the sp4 codes.

- **wcvp_matching_zotz**  
  Script to add WCVP names to the Zotz list.

- **zchecknewfile.r**
