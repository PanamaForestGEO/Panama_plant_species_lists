# Files that were previously used in the repository that are now out of use.


- **zarchive/**: Deprecated or legacy versions of species lists.

- **labelbox_lists** contains species and complete (species, genera and families) lists for Labelbox project with GBIF taxon ID.  It has its own readme.

- **tocheck** contains files generated to enable manual / other checks of potentially problematic records.

- **splist_mid** contains files that used to be work as intermediate products for taxonomic resolution but are now out of use. 

- **scripts** contains scripts that are now out of use and have not been updated.
