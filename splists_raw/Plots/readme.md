# Species recorded in plots

This subdirectory contains files of plot censuses in BCI and in other sites of the BCNM: 

- **abundance_plots_HM.xlsx**: this file contains abundance of trees in different plots of BCNM.

- **doi_10_5061_dryad_1g1jwsvc3__v20260226**: this subdirectory contains the files from the publication of large tree census in 58ha of BCI (Dataset DOI: 10.5061/dryad.1g1jwsvc3) 

- **gigante_census.csv** List of species codes and abundance in the Gigante project. This file was derived from the file "Gigante_2023census_WorkingFile20240829" which is not publicly available at the moment.
